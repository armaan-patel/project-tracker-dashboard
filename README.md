# Project Tracker Dashboard

> A feature-rich, single-file project tracker with multiple chart types, dark mode, and a custom accent color picker — no install or build step required.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

---

## Features

- **Status filters** — view All, In Progress, Completed, or Unfinished projects at a glance
- **4 chart types per project**
  - **Progress** — slider-controlled percentage with an animated donut chart
  - **Burndown** — log daily remaining work; ideal vs. actual line chart
  - **Gantt** — task timeline with start/end dates and a live "Today" marker
  - **Milestones** — checklist of key checkpoints with target dates and a completion bar
- **Notes** — per-project freetext notes that auto-save as you type
- **Dark mode** — toggle between light and dark themes; preference is remembered across sessions
- **Accent color picker** — 10 preset colors + a custom hex picker; updates the entire UI instantly
- **Persistent storage** — all data saved in browser `localStorage`; no backend or account needed

---

## Usage

1. Download or clone this repo
2. Open `project-tracker.html` in any modern browser
3. Click **+ New Project** to get started

No installation, no dependencies, no build step required.
